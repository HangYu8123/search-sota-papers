# General list (147)

The always-in-force part of the `institutions: reputable` accept-list. Read this
whenever the institution gate runs — every resolution path includes it. Topic
files beside this one are **additive**; none of them replaces or narrows this
list.

Procedure, provenance, the affiliation ladder, the matching rules, and the
outcome table live in `../institutions.md`. This file is only the list.

> **Two measurement bases, one flat accept-list.** Tiers 1–4 come from
> AI-specific signals (CSRankings-AI, per-venue affiliation shares across
> NeurIPS/ICML/ICRA/RSS/CoRL, ML/CV/NLP/robotics volume). *Broad research
> excellence* comes from general research indices (ARWU, THE, QS, US News,
> Nature Index, Leiden). The grouping records **where an entry's reputation was
> measured** and carries no precedence: for the gate, all 147 entries are one
> flat list. Never compare a tier number against a broad-list position as if
> they were the same scale, and never report either to the user as a published
> rank — no rank here is independently verified.
